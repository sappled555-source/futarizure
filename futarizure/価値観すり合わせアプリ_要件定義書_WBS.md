作成日: 2026-07-18 / 版: v3.0(フェーズ4着手: オンライン対戦+PWA化)

## 0. 改訂履歴

| 版 | 変更内容 | 決定者 |
|---|---|---|
| v1.0 | 参考アプリ準拠(秘密の数字を言葉で表現し昇順に並べる協力ゲーム) | - |
| v2.0 | **コア体験を変更**: お題に対して各自がテキストで自由回答→公開→ズレを話し合う方式へ | 使用者 |
| v2.1 | 答え合わせを一斉公開→**1人ずつタップでめくる方式**に変更。一致/ズレ判定・スコアは**廃止**(ゴールは価値観の可視化と話題づくりのため) | 使用者 |
| v3.0 | ユーザーテスト好感触につきフェーズ4着手。**オンライン対戦とアプリ化を同時進行/無料枠で検証/まずPWA→反応を見てストア配信** | 使用者 |

## 1. プロジェクト概要

| 項目 | 内容 |
|---|---|
| 目的 | パートナー/グループの価値観のズレを、お題への自由回答を通じて解明する |
| コア体験 | 例) お題「一番記憶に残っているデート場所は?」→ 彼「花火大会」/ 彼女「夜中にダル着でコンビニにアイス」→ 答え合わせで盛り上がる |
| 方針決定事項 | ①まずWebで検証→後でアプリ化 ②1台パス回し(サーバー不要) ③カップル・夫婦がメイン |
| MVPの形態 | ブラウザで動く単一HTMLファイル |

## 2. 機能要件(MVP)

| ID | 機能 | 内容 | 優先度 |
|---|---|---|---|
| F-01 | プレイヤー設定 | 2〜6人の名前を登録(デフォルト2人) | 必須 |
| F-02 | お題カテゴリ選択 | 思い出/お金/暮らし/将来/気持ち + カスタムお題入力 | 必須 |
| F-03 | 秘密のテキスト回答 | パス回しで各自が回答を入力(他の人には見えない) | 必須 |
| F-04 | 順次公開 | 回答カードを1人ずつタップしてめくる | 必須 |
| F-05 | (廃止) 一致/ズレ判定・スコア | v2.1で廃止。判定せず会話に集中する設計 | 廃止 |
| F-06 | 話し合いプロンプト | カテゴリに応じた深掘り質問を提示 | 必須 |
| F-07 | (廃止) スコア表示 | v2.1で廃止 | 廃止 |
| F-08 | 連続プレイ | 同カテゴリで次のお題へ(お題は使い切りローテーション) | 推奨 |
| F-09 | オンライン対戦・回答の保存 | 遠隔プレイ/履歴保存 | 対象外(次フェーズ) |

## 3. 非機能要件

- スマホ縦画面最適化 / サーバー・登録・通信不要 / 個人情報を収集しない
- 覗き見防止: 入力後は回答を隠して次の人へ渡す導線

## 4. 画面一覧

1. ホーム 2. プレイヤー登録 3. カテゴリ選択(+カスタム) 4. お題発表 5. 回答入力(パス回し) 6. 答え合わせ(1人ずつめくる) 7. 話し合いプロンプト
